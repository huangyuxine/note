# 常用命令

### 查看进程 <a href="#cha-kan-jin-cheng" id="cha-kan-jin-cheng"></a>

`pa -aux | grep mqtt`

### 查看端口

`netstat -ano | grep 1883`

### 干掉进程

`kill -9 ID`

### 后台运行脚本

`nohup php -f /www/wwwroot/3.com/server.php &`

### 删除日志

`find /www/wwwlogs -name "*.log" -exec rm -rf {} \;`
